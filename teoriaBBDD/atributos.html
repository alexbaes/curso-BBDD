Qué es un atributo

En una base de datos, un atributo se refiere a una característica específica o propiedad de una entidad. Los atributos describen las cualidades o características de las entidades y son utilizados para almacenar y representar información.

Cada entidad en una base de datos tiene uno o más atributos que definen sus propiedades. Por ejemplo, en una base de datos de empleados, los atributos de la entidad "Empleado" podrían ser el nombre, apellido, número de identificación, fecha de contratación, salario, etc.

Los atributos tienen un tipo de dato asociado que define el tipo de información que se puede almacenar, como texto, números, fechas, entre otros. Además, los atributos pueden tener restricciones, como valores únicos, valores requeridos o límites de tamaño.

Tipos de atributos

Los tipos de atributos a nivel de programación se convierten en los tipos de datos que podemos almacenar

  • Números enterosy flotantes
  • Cadenas y carácteres de texto
  • Fechas y horas
  • Booleans
  • Blobs y archivos
  • Datos geográficos
